Implementa algunas mejoras en esta aplicación, que utiliza el API de recetas de la web www.recipepuppy.com, con Ruby on Rails.

Para ello, clona este repositorio para utilizarlo como base y amplía la aplicación, por ejemplo con:

- Bootstrap Añadido
- funcionalidad JavaScript añadida
- base de datos para usuarios
- Añadido un modelo para usuario y gestionar su acceso
- Permitir que los usuarios registrados busquen nuevas recetas.
